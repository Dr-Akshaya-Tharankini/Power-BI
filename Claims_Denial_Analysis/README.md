# Healthcare Claims & Denial Analytics Dashboard (Power BI)

## Overview
This interactive **Power BI dashboard** provides data-driven insights into **healthcare claims performance**, helping administrators, physicians, and payer management teams identify **claim submission trends, denial patterns, and payer performance metrics**.

The dashboard is designed to **optimize revenue cycle efficiency** by highlighting key metrics such as **Claim Count, Claimed Amount, Denied Amount, Rejection Rate, and Top Denial Reasons** ‚Äî empowering healthcare organizations to take timely, evidence-based decisions.


## Key Insights Delivered
- **Total Claim Volume** ‚Äì Track claim submissions and processing trends  
- **Claimed vs Denied Amounts** ‚Äì Measure total value and proportion of denied claims  
- **Rejection Rate** ‚Äì Monitor performance and identify inefficiencies  
- **Top Payers by Rejection %** ‚Äì Spot payers contributing to higher denials  
- **Top Denial Codes & Services** ‚Äì Understand common denial causes and affected services  
- **Denial Trends by Month & Payer** ‚Äì Visualize denial fluctuations over time  
- **Denial Reasons by Provider** ‚Äì Detect recurring operational or documentation issues  
- **Rule-Based Recommendations** ‚Äì Actionable guidance for denial prevention and process improvement  


## Dashboard Sections

### 1. Claims Summary
- KPIs: **Claim Count, Claimed Amount, Denied Amount, Rejection Rate, and Active Payers**  
- Filters for **Month, Department, Payer, and Provider**  
- Auto-refresh timestamp for latest data visibility  

### 2. Denial Analytics
- **Top Payers ‚Äì Rejection %:** Identifies payers with the highest denial ratios  
- **Top Denial Codes:** Highlights denial frequency by reason codes  
- **Top Denied Services:** Visualizes services most affected by denials  
- **Payers ‚Äì Denial Amount:** Shows financial impact per payer  

### 3. Denial Trends & Provider Analysis
- **Denied Amount by Month & Payer:** Tracks financial trends of denials  
- **Top Denial Reason by Provider:** Shows how denial patterns vary across physicians  
- **Weekly Denial Heatmap:** Visualizes denials by weekday to find operational gaps  
- **Rule Recommendations Table:** Provides corrective actions and expected financial impact
