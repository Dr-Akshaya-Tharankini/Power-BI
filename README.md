# Interactive Healthcare Analytics Dashboard – Power BI

**Data that saves lives — and money.**  
This project showcases an **Interactive Healthcare Dashboard** built in **Power BI** to transform raw hospital data into actionable insights.

---

## Project Overview
Hospitals generate huge amounts of data daily — from patient visits and revenue to provider ratings and departmental performance. This dashboard helps stakeholders monitor **real-time KPIs**, track trends, and make informed decisions in minutes, not months.

---

## Key Features

### 1. Provider Insights
- Compare average ratings, patient visits, and department contributions over time

### 2. Revenue Analysis
- Track YTD vs PYTD performance with variance %
- View revenue splits by service type and department

### 3. Hospital Overview
- Real-time KPIs on total revenue, pending payments, and patient demographics
- City and state-level patient volume analysis

### 4. Drill-Down & Filters
- Slice by year, month, gender, provider, department, or city

### 5. Clear Visual Storytelling
- Heatmaps, bar charts, line graphs, and dynamic KPI cards for quick decision-making

---

## The Impact
✅ Real-time visibility into hospital operations  
✅ Faster decision-making for administrators  
✅ Improved tracking of growth and performance trends  

---

##  Tech Stack
- **Power BI Desktop**
- **DAX (Data Analysis Expressions)**
- **Power Query**
- **Time Intelligence Functions (YTD, PYTD)**
- **Row-Level Security (RLS)**
